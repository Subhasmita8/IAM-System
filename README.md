📌 Overview

The IAM System is a full-stack web application designed to manage user authentication, authorization, roles, and permissions. It implements Role-Based Access Control (RBAC) to ensure secure and controlled access to system resources.

Built using React (frontend), JavaScript (backend), and SQL (database), this project demonstrates core identity management concepts used in modern enterprise applications.
